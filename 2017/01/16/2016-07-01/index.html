ubuntu 添加coretemp 模块

coretemp本来是ubuntu系统自带的可以读取thermal sensors的模块文件，但是在Xen Guest VM中，却不会自动包含coretemp模块，因此需要自己手动编译，加载coretemp模块。

步骤:

  1. 安装依赖的包：apt-get install build-essential gcc linux-headers-uname -r
  2. 创建一个工作文件夹：
    mkdir coretemp
cd coretemp
  3. 从Ubuntu Kernel下载比较新的kernel源代码，然后从中抽取coretemp.c文件
  4. 编写Makefile文件（注意：Makefile文件M必须大写）
  5. 命令行输入make即可以生成coretemp.ko文件
  6. 加载coretemp.ko模块：insmod coretemp.ko

PS：需要注意的是下面链接给的coretemp.c文件因为版本过老不能使用，所以应注意#3步骤
